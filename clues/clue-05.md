# 🔍 Clue 05 - Count the Evidence

How many suspicious actions did the thief take?

## 🧩 Your Task

Count how many times "alex" appears in the log file!

## 💡 Hint
Combine two commands using a pipe "|"
One command searches, the other counts lines...

## ✅ Command to Learn
grep "alex" /mystery/room4/server.log | wc -l

Two crimes committed at midnight by the same person!
