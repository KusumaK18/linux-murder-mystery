# 🔍 Clue 01 — The Hidden Files

The thief always hides in plain sight...
Hidden files in Linux start with a dot (.)

## 🧩 Your Task

Run this command on your Linux machine:
mkdir -p /mystery/room1
cd /mystery/room1
touch .hidden_clue

Now list ALL files including hidden ones.

## 💡 Hint
Normal `ls` won't show hidden files.
You need a special flag...

## ✅ What you should see
A file called `.hidden_clue` appears!

## 🔑 Command to Learn
ls -la

## ➡️ What's inside .hidden_clue?
The suspect was last seen near **/mystery/room2**
