# 🔍 linux-murder-mystery
Solve the mystery using only Linux commands! 

> A file has gone missing from the server at midnight...
> Someone deleted the evidence. Your job is to find the culprit
> using ONLY Linux commands!

## 🕵️ The Story

A secret file called `secret.txt` was stolen from the server.
The thief left behind clues — but they're hidden deep inside
the Linux filesystem. Can you find who did it?

## 🎮 How to Play

1. Spin up a Linux machine (AWS EC2 or any Linux terminal)
2. Follow the clues one by one starting from clue-01
3. Each clue teaches you a real Linux command
4. Solve all 5 clues to reveal the culprit!
